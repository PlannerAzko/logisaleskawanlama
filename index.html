🛠️ Panduan Setup Google Apps Script

×

Metode ini memungkinkan siapa saja (dari berbagai email) untuk klik submit dan datanya otomatis masuk ke Spreadsheet & Drive yang sama milik Admin tanpa perlu login akun Google di aplikasi!

Langkah-langkah Setup:

  1. Buka Google Spreadsheet baru.
  2. Di menu atas spreadsheet, pilih Extensions (Ekstensi) > Apps Script.
  3. Hapus semua kode default di dalam editor script, lalu salin dan tempel kode di bawah ini.
  4. Klik tombol Save (Simpan) 💾 di bagian atas.
  5. Klik tombol Deploy (Terapkan) di kanan atas > pilih New deployment (Penerapan baru).
  6. Pilih jenis Web app (Aplikasi web) (klik ikon gerigi di sebelah kiri "Pilih jenis" jika belum muncul).
  7. Ubah konfigurasi berikut:
    • Execute as (Jalankan sebagai): Me (Email admin Anda)
    • Who has access (Siapa yang memiliki akses): Anyone (Siapa saja) <-- PENTING agar semua user dapat submit!
  8. Klik tombol Deploy, lalu berikan izin akses (Authorize access) jika diminta oleh akun Google Anda.
  9. Salin Web App URL yang dihasilkan (berakhir dengan /exec).
  10. Klik tombol "Atur/Ubah URL Web App" di aplikasi ini, lalu paste URL tersebut dan klik Simpan!
