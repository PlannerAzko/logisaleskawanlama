🛠️ Panduan Setup Google Apps Script

×

Metode ini memungkinkan siapa saja (dari berbagai email) untuk klik submit dan datanya otomatis masuk ke Spreadsheet & Drive yang sama milik Admin tanpa perlu login akun Google di aplikasi!

Langkah-langkah Setup:

  1. Buka Google Spreadsheet Anda.
  2. Di menu atas spreadsheet, pilih Extensions (Ekstensi) > Apps Script.
  3. Hapus semua kode default di dalam editor script, lalu salin dan tempel kode lengkap di bawah ini.
  4. Klik tombol Save (Simpan) 💾 di bagian atas editor.
  5. ⚠️ PENTING (Otorisasi Izin Akses):
    Sebelum melakukan deploy, di bagian atas editor Apps Script terdapat dropdown pilihan fungsi (biasanya tertulis doPost).
    - Ubah dropdown tersebut menjadi triggerAuth.
    - Klik tombol Run (Jalankan) di sebelahnya.
    - Klik Review Permissions (Tinjau Izin) saat pop-up muncul.
    - Pilih akun Google Anda, klik Advanced (Lanjutan) > klik Go to ... (tidak aman) > klik Allow (Izinkan).
    Langkah ini wajib agar Google memberikan izin kepada script Anda untuk mengunggah file ke Google Drive dan menulis data ke Spreadsheet!
  6. Klik tombol Deploy (Terapkan) di kanan atas > pilih New deployment (Penerapan baru).
  7. Pilih jenis Web app (Aplikasi web) (klik ikon gerigi di sebelah kiri "Pilih jenis" jika belum muncul).
  8. Ubah konfigurasi berikut:
    • Execute as (Jalankan sebagai): Me (Email admin Anda)
    • Who has access (Siapa yang memiliki akses): Anyone (Siapa saja) <-- PENTING agar semua user dapat submit!
  9. Klik tombol Deploy.
  10. 💡 PENTING (Setiap Kali Update Kode):
    Setiap kali Anda mengubah/memperbarui kode di Apps Script editor, Anda WAJIB membuat versi baru!
    - Klik Deploy > Manage deployments.
    - Klik ikon Pensil (Edit) di kanan atas.
    - Pada dropdown "Version", pilih "New version" (Versi baru).
    - Klik Deploy, lalu salin kembali URL terbaru (biasanya tetap sama, namun versi baru wajib di-deploy agar perubahan kodenya aktif!).
  11. Salin Web App URL yang dihasilkan (berakhir dengan /exec).
  12. Klik tombol "Atur/Ubah URL Web App" di aplikasi ini, lalu paste URL tersebut dan klik Simpan!
